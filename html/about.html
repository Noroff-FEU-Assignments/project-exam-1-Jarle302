About

Purpose

This blog is made as my exam project first year studying front end development at noroff. I could make a blog about anything, i decided to create a automatic blog, so i first made a backend app that creates blog posts and makes images that fits the content without having to be manually prompted.

How it works

The blog is generating the posts using openAis api, where it gets prompted to create a blog post. As part of the process, the API also dynamically creates images that fit the text of the post. The generated content is then posted to WordPress, which serves as a headless CMS for our blog. The front end of the website fetches posts from the WordPress API to display them in a blog format. Additionally, our backend is set to automatically create a new post every day on a timer, providing a steady stream of fresh content for our readers all by itself."
